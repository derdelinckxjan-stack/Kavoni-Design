## Brand Voice

| Attribute | Description | Example |
|---|---|---|
| **Confident** | We know our products and our customers | "Engineered for the M3. Built for you." |
| **Precise** | Specific fitment, specific results | "Fits F80 M3 2014–2018, S55 engine" |
| **Premium** | Luxury tone without being inaccessible | "Carbon fibre. Not carbon look." |
| **Purposeful** | Every word earns its place | Short sentences. No filler. |
| **Passionate** | We love BMWs too | "This is the diffuser the F82 deserves." |

**Avoid:** Excessive exclamation marks, vague superlatives ("best quality!", "amazing product"), generic descriptions that could apply to any car.

## Product Description Templates

### Template A — Aero / Body Parts
```
[PRODUCT NAME] — [MODEL] [CHASSIS]

Engineered specifically for the [chassis code] [model name], this [product type] delivers [key benefit 1] and [key benefit 2] without compromising the [OEM/daily driver] balance.

**Fitment:** [Chassis code] · [Year range] · [Body styles]
**Material:** [Carbon fibre / ABS / Polyurethane]
**Finish:** [Gloss black / Matte / Unpainted]
**Mounting:** [OEM clip-in / Adhesive / Bolt-on]

Compatible with: [specific variants — e.g. M Performance bumper / standard bumper]

In the box: [list contents]
```

### Template B — Exhaust Systems
```
[PRODUCT NAME] — [MODEL] [CHASSIS]

[Engine code] owners: this is the exhaust your [model] was waiting for. [Key sound/performance claim — be honest].

**Fitment:** [Chassis code] · [Year range] · [Engine(s)]
**Type:** [Cat-back / Axle-back / Valved]
**Material:** [304 stainless steel / Titanium]
**Tips:** [Size] · [Finish]
**Sound level:** [Street / Sport / Race]

Professional installation recommended. Does not affect manufacturer warranty in most cases — consult your dealer.

In the box: [list contents + hardware]
```

### Template C — Lighting
```
[PRODUCT NAME] — [MODEL] [CHASSIS]

A direct replacement for [OEM headlights/taillights] on the [model], bringing [LED / sequential / DRL upgrade] to an already sharp design.

**Fitment:** [Chassis code] · [Year range]
**Type:** [Full assembly / Lens only / DRL strip]
**Technology:** [LED / OLED / Halogen housing]
**CANBUS:** [Compatible / Resistor required]
**Coding:** [Not required / May require ISTA coding — noted]

Plug-and-play on most variants. [Note any exceptions.]

In the box: [pair / set · hardware · instructions]
```

### Template D — Interior / Controls
```
[PRODUCT NAME]

Upgrade your [steering wheel / gear knob / trim panel] with a part that [key benefit — feel, look, precision].

**Fitment:** [Universal / Model-specific — list chassis]
**Material:** [Alcantara / Leather / Carbon fibre / Aluminium]
**Colour options:** [if applicable]

[Safety note for steering wheels:] Installation requires disconnecting the battery and allowing airbag capacitors to discharge. Kavoni Design recommends professional installation by a qualified BMW technician.

In the box: [contents]
```
